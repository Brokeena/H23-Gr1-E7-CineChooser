# H23-Gr1-E7-CineChooser

CineChooser is a mobile application that recommends movies to you and your friends, helping you find a movie that suits everyone. The app is based on a swipe model, which makes it simple and intuitive to use. It is currently being developed as part of an integration project in Computer Science and Mathematics at Collège de Bois-de-Boulogne.

## The project works on Android (see APK) or on an emulator in Android Studio
